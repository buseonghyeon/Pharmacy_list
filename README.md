### 📖 프로젝트 개요
WaterMelon은 JavaScript로 구현된 프로젝트로, 약국 위치 및 정보를 지도 상에 표시하는 서비스를 제공합니다.
<br/><br/>

### 🚀 프로젝트 목표
- 사용자의 현재 위치를 기반으로 가까운 약국을 지도에 표시
- 약국의 이름, 주소, 전화번호 및 영업시간을 제공
- Naver Maps API와 공공데이터포털 API를 사용하여 약국 정보를 가져옴
<br/><br/><br/>
